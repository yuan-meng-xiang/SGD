Reference 0004

知识图谱学习法速查卡

工具性学科不要先画完整百科，也不要只逐章堆碎片。用薄框架导航，用小课扎根，用阶段图谱重组。

一句话

先给 5-7 个主干节点，再一课一课长出细节，每 3-5 课重画一次当前地图。

学习节奏

薄框架 小课 练习 可选追问 learning record 关系更新 阶段图谱

可选苏格拉底追问

学习者先完成课程最后留下的问题；老师先评价和打磨这份回答，再询问是否开启苏格拉底追问。不开启时，课程按普通短课结束；开启时，只追问 2-3 个关键问题，帮助学习者把模糊回答打磨成可解释、可迁移、可连接到图谱的理解。

先问是否开启，再追问；先看到理解变化，再写 learning record。

learning record 要多写一层

以前只写 以后还要看 目的
学会了什么 它连到哪几个旧概念 避免知识点孤立
用户怎么证明理解 它改变了哪一个旧理解 记录认知升级
下一课可以教什么 它暴露了哪个缺口 判断最近发展区

阶段复盘四问

  • 有没有孤立节点，只会背名词但不知道它连到哪里？
  • 有没有重复节点，其实是在说同一个概念？
  • 有没有关系不清，导致真实任务里不知道先做什么？
  • 有没有时间敏感内容，需要回到官方资料重新确认？

当前 Codex 应用

Codex 的当前薄框架已经启动：定位、入口、工作流、上下文、执行与验证、扩展能力、安全边界。阶段快照见 Codex 当前知识图谱。追问类型见 可选苏格拉底追问卡。
