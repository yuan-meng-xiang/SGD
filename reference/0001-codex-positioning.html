Codex 定位速查卡

给别人讲 Codex 时，先讲清它的角色，再讲功能入口。

一句话

Codex 是可以进入项目现场、推进软件工程任务到可 review 状态的 AI coding agent。

三层讲法

初级

它能帮你写代码，但这不是重点。

中级

它能读项目、改文件、跑命令、看 diff、补测试。

高级

它能按项目规则、上下文和验收标准，完成一项可控的工程任务。

对比句

  • ChatGPT 更像顾问，帮你想清楚；Codex 更像执行者，帮你推进任务。
  • Cursor 更像 AI 编辑器，陪你改代码；Codex 更像工程 Agent，跑完整任务。
  • Claude Code 更偏终端长期协作；Codex 更偏 OpenAI 生态里的多端任务执行。

适合任务

目标明确、范围可控、上下文清楚、结果能验证、失败能回滚、风险可接受。

不适合直接交给它

生产数据库、真实用户数据、支付核心逻辑、权限安全核心模块、大规模架构迁移、你自己无法验收的任务。

Sources: `Codex橙皮书.pdf` pages 5-15; official details should be checked against OpenAI Codex documentation when teaching current product behavior.
