Reference 0003

Codex App 第一次任务速查卡

第一次上手不追求做大项目，只追求跑完一条完整闭环。

一句话

选一个安全练习项目，开一个清楚 thread，给一个小任务，看 diff 后再决定是否接受。

关键概念

概念 给新手的讲法 讲解重点
项目目录 Codex 要进入哪个文件夹工作 不要第一次就选重要项目或整个磁盘
项目列表 Codex 能看到的不同代码项目 不是聊天记录列表，而是代码项目列表
thread 同一个项目里的一个具体任务对话 一个明确任务开一个 thread
任务窗口 给 Codex 下指令和看执行过程的地方 能看到任务、计划、执行、总结和后续输入
review pane 检查 Codex 改了什么的地方 不要只看总结，要检查实际改动
diff 新增和删除的代码对比 每次完成任务后先看 diff，再决定是否接受

第一次流程

1. 登录 2. 选练习项目 3. 开 thread 4. 给小任务 5. 看计划与执行 6. 看 review pane 7. 看 diff 8. 决定是否继续

第一个任务

请帮我做一个简单网页，黑色背景，中间显示大字 Hello, Codex，字体白色，页面整体水平和垂直居中，只使用 HTML 和 CSS。
