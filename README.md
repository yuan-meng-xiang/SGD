# SGD

Codex learning trail based on `Codex橙皮书.pdf`.

This repository stores a stateful learning workspace for studying and teaching Codex:

- `MISSION.md` records the learning mission.
- `RESOURCES.md` records trusted sources and reference material.
- `lessons/` contains short HTML lessons.
- `reference/` contains quick-reference cards.
- `learning-records/` captures demonstrated learning progress.
- `knowledge-graphs/` contains stage snapshots that connect lessons and learning records into a growing map.
- `assets/` contains reusable lesson components.

The current goal is to learn Codex well enough to explain its background, product shape, and practical usage to other people.

The learning method is: start with a thin framework, teach one small concrete skill per lesson, record demonstrated understanding, and redraw the current knowledge graph every 3-5 lessons.

After a new lesson is created, Socratic follow-up can be started as an optional mode. It is used to sharpen the learner's answer through a few focused questions before any learning record or knowledge-graph update is made.
